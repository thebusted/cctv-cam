## Summary and Market Analysis (English Version)

### Scope of Work

**Development of Web-Based Dashboard covering 6 systems:**

1. **HVAC** (Air Conditioning System)
2. **Pump** (Water Pump System)
3. **Lighting** (Electrical Lighting System)
4. **Gate System** (Automatic Gate)
5. **Lift** (Elevator System)
6. **CCTV** with Face Detection/Recognition (4 cameras)

**Face Detection/Recognition Module:**

* Receive RTSP streams from client-provided cameras
* Process with YOLO + Python
* Train/register faces on GPU (development/local)
* Real-time detection runs on cloud CPU
* Display live results on dashboard

**Cloud Server:**

* Supports all 6 systems + 4 cameras
* Spec: 2–4 vCPU, 4–8GB RAM, 50–100GB storage
* Cost: **1,500–2,000 THB/month**

**Special Conditions:**

* Hardware (cameras, servers) provided by client
* No on-site installation fees
* No Video Wall setup fees
